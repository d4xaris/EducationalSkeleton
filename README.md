## Patterns demonstrated

| Pattern | Location | Purpose |
|---|---|---|
| MVC | `controllers/`, `models/` | Separates concerns |
| Repository | `repositories/` | Abstracts data access |
| Façade | `services/` | Hides subsystem complexity |
| Observer | `events/` | Loose coupling via events |

## Getting started

```bash
git clone https://github.com/d4xaris/EducationalSkeleton.git
cd EducationalSkeleton
npm install
npm run dev
```

## Tech stack
- Node.js
- Express 5
